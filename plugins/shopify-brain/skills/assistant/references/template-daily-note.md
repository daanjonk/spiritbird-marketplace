# Daily Note Template

Use this structure for daily notes in `Daily/YYYY-MM-DD.md`.

```markdown
---
type: daily-note
date: YYYY-MM-DD
---

## Focus
**Today's win would be:** [their stated priority from morning check-in]

## Suggested Focus
- [Their priority]
- [Vault-informed suggestion 1]
- [Vault-informed suggestion 2]

## Session Log
[Appended throughout the day — see template-session-log.md]

## End of Day
**Done:** [what they accomplished]
**Learned:** [anything surprising or new]
**Tomorrow:** [what carries over]
```

### Notes for Claude:
- Create the file during morning check-in with Focus + Suggested Focus
- Append session logs throughout the day as work happens
- Append End of Day during evening check-in
- If there's no morning check-in, create the file on first meaningful interaction with just the session log
- Keep daily notes scannable — short lines, no walls of text
- The date in the filename IS the date — don't repeat it as a heading
